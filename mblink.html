Installation et configuration de MumbleLink

Voici un tuto pour installer le MumbleLink pour le KTA.
Si vous n'avez pas Mumble vous pouvez le télécharger ici : http://www.mumble.com/mumble-download.php


Téléchargez Minecraft Forge en version "Download Recommanded" :
files.minecraftforge.net
Puis démarrez-le et faites "install client"
Ensuite téléchargez le MumbleLink :
www.minecraftforum.net/forums/mapping-and-modding/minecraft-mods/1272675-1-8-mumblelink-forge-smp-lan-mumble-realism
Puis mettez le jar dans votre dossier "mods" (disponible dans votre dossier ".minecraft" inclus dans %appdata%)
Vous devez ensuite configurer le MumbleLink :
Allez sur votre Mumble puis Configurer>Paramètres, activez "Avancé" (en bas à gauche)
Allez dans "Plugins" et activez "Lier au jeu et transmettre la position"
Allez dans "Sortie audio" puis dans "Position Audio", configurez comme ceci
- Distance minimale : 1.0 m
- Augmentation : 150 %
- Distance maximale : 40.0 m
- Volume minmum : 0 %

remiber, ce putain d'génie
